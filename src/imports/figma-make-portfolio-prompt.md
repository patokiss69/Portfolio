# Prompt optimizado para Figma Make

**Prompt:**

Design a **modern, minimal and professional portfolio website layout** for a **Senior UX/UI Lead and Product Designer with 20+ years of experience**.

The portfolio should be designed for **international recruiters, product companies and remote freelance opportunities**.

The design must look similar to **top-tier product designer portfolios built in Framer or Webflow**.

Use a **clean editorial layout with strong typography, lots of whitespace and a product-design aesthetic**.

---

# Layout & Grid

Create a responsive desktop layout with:

* **1440px canvas width**
* **12 column grid**
* generous margins and whitespace
* vertical rhythm between sections
* modular spacing system

Use a **clean grid-based structure typical of modern product design portfolios**.
